Introduction to HTML

HTML is a web designing language which stands for Hyper Text Markup Language. The latest version of HTML as of March 2019 is HTML 5. HTML includes a language called CSS. HTML is a very fun language. C'mon I'll show you.

What you need to know
This guide assumes that you have the following:
  • Some knowledge in programming
  • Able to recognize English words and knowledge of grammar
  • Able to have fun!!

Let's get started

Saving and opening projects

Windows

  1. Open Notepad or any HTML editor
  2. Go to File
  3. Click the Save Button
  4. Windows will ask you to put in a name.
  5. Put in a a name- For this we will use "project.html"
    6. **Note: The .html part is neccesary in order for your file to run as a website
  6. Click the save button

Mac

  1. Open Text-Edit or any html editor
  2. Click Format
  3. Click Make Plain text
  4. Click File
  5. Click Save As
  6. Enter a name - For this we will use "project.html"
    **Note: The .html part is neccesary in order for your file to run as a website
  7. Click save
  8. An error box will ask you if you want to keep it as a html file- Click"Use .html"
Hello World- Your first project

In your HTML editor or notepad write:

<p>Hello World</p>

Open your file in your browser and watch your website come to life!

Inserting images

Inserting images is really easy. The image tag is used when inserting images into your webpage. They have a specific opening and closing tag. Here is the code:
<img src="Your source to the image" alt="Alternate text"> The alternate text is regquired incase your image doesn't load properly. It is also used for screen readers can't see the images but will still see the content of it. In you project go ahead and add an image. Remember how? No? That's ok. Just look up. You can add whatever image you want. Just make sure you have an alt text and a valid source

Inserting comments

Inserting comments is really easy to. Comments are just used to simplify your code for anybody else who looks at your code. Adding a comment is done like this:
<!-----Comment about the code---->
Some comment titles could be Introduction section, Nav-bar. Comments are not seen by the auience of your wbsite. In your website go ahead and add a comment to your code. Make one for your header and your paragraph tag.

Anchoring links
<> Links are easy to insert. They help you link to another page within or outside your webpage on a new tab. Here is the code:

<a href="The webpage"> Link name</a>or <a href="The webpage" target="_blank"> Link name </a>

As you can see there are two versions of inserting links. The first is a internal link meaning it opens the link in the same tab. The other is a external link meaning it opens in another link. To achieve the external links you must put target="_blank" Here is a link to my profile with external and an internal:
